# Hero Reference Notes

## UI / UX reference
The visual direction is inspired by a minimal landing page hero:
- bright off-white or white background
- slim top navigation
- centered headline and supporting copy
- one primary CTA in the middle
- large negative space
- monochrome or near-monochrome hero artwork
- clean premium startup/editorial feeling
- a subtle row of logos or social proof near the bottom is optional

## Motion reference
The motion direction is inspired by a Pinterest video reference:
- halftone dots appear to sweep horizontally
- the movement feels like a scan, reveal, or controlled wave
- the motion is smooth and premium, not flashy
- dots should feel attached to the subject shape
- the reveal should read clearly from left to right

## Desired adaptation for this project
Use the two-hands image already inside the project as the subject source.
